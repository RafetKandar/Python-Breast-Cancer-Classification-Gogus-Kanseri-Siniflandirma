# Python-Breast-Cancer-Classification-Gogus-Kanseri-Siniflandirma

### eng : Breast Cancer Classification --> Using the Knn algorithm, it detects whether the tumor is benign or malignant in people diagnosed with breast cancer. I got the dataset from <a href="https://www.kaggle.com/uciml/breast-cancer-wisconsin-data">Kaggle </a> It contains 596 rows and 32 columns of tumor shape and specifications.</br>
